\subsection{Detailed Description}
\hyperlink{classWarehouse}{Warehouse} Class. 

This \hyperlink{classWarehouse}{Warehouse} class defines a warehouse that will keep track of the all clients, packages, and trucks. It will also generate the routes for all the trucks and load the trucks with packages.
